# Log de self-reviews semanales

Registro automático de cada ejecución de `src/self_review.py` tras publicar una edición. Solo se anotan aquí las ejecuciones con **algún score < 7** (las que dispararon alerta). El resto quedan solo en `private/self-review/YYYY-wWW.md` individual.

**Alertas Telegram asociadas:**

- Score < 7 en cualquier dimensión → alerta urgente con link al archivo detallado.
